How to install:  
Extract to AddOns folder. Rename and remove "-main" from the folder RickeBuffSize-main  

How to customize settings:  
/buffsize <10-60> #change the size of buffs on target  
/buffsizeplayer <10-60> #change the size of your buffs on target  
/debuffsize <10-60> #change the size of other players debuffs on target  
/debuffsizeplayer <10-60> #change the size of your debuffs on target
